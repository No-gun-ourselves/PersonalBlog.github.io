搭建好hexo之后，如何发布、删除、修改个人blog？

#发布
1、在自己创建的博客目录下，shift+鼠标右键点击文件夹，用PowerSheel命令行进入，按照下面顺序，依次创建、生成、预览（如果不需要预览，则可以省略这一步）、部署。

​

2、hexo n “我的博客” == hexo new “我的博客” //新建文章

3、hexo s == hexo server //启动服务预览博客

4、hexo g == hexo generate //生成博客

5、hexo d == hexo deploy //部署博客

#删除

1、在个人博客目录下，进入source/_post,找到对应的.md文件，删除它

2、依次执行hexo g,hexo d命令

3、若出现404，稍等一下，再次查看网站即可
